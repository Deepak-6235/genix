## Image Recommendations:

- **Format**: JPG or WebP (WebP recommended for better compression)
- **Size**: Optimize images before uploading (use tools like TinyPNG or ImageOptim)
- **Dimensions**: 
  - Hero background: 1920x1080px or larger
  - Service images: 600x400px
  - Blog images: 800x500px
  - About us: 800x600px
